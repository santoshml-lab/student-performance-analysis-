# Student Performance Analysis

This project analyzes student performance based on their behavior and predicts outcomes using a machine learning model. The goal is to understand patterns in student performance and evaluate the model's predictive accuracy.

## Dataset

- The dataset contains 500 records of students with features such as review counts, engagement metrics, and behavior indicators.
- The target variable is student performance (Pass/Fail or grade prediction depending on your model).

## Analysis & Visualizations

### Confusion Matrix
The confusion matrix shows how well the model performed in classifying students correctly. It highlights the number of true positives, true negatives, false positives, and false negatives.

### Key Insights
- The model achieved high accuracy in predicting student performance.
- Most students fall into the middle performance range, with fewer extreme high or low performers.
- Behavioral patterns such as engagement and review counts are strong indicators of performance.
